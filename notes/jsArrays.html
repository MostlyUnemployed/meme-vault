Arrays and array.methods()

Arrays are JavaScript's way of making lists. These lists can store any data types (including strings, numbers, and booleans) and they are ordered, meaning each item has a numbered position.
Something important to note is that JavaScript starts counting from 0 rather than 1, so the first item in an array will be at position 0. This is because JavaScript is zero-indexed. 


    let condiments = ['Ketchup', 'Mustard', 'Soy Sauce', 'Sriracha'];
    const utensils = ['Fork', 'Knife', 'Chopsticks'];
    console.log(condiments);
    console.log(utensils);
    utensils = ['Memes'];
    console.log(utensils);
      
      [ 'Ketchup', 'Mustard', 'Soy Sauce', 'Sriracha']
      [ 'Fork', 'Knife', 'Chopsticks' ]
      [ 'Memes' ]

Variables that are assigned with const cannot be reassigned. However, arrays that are declared with const remain mutable, or changeable. This means that we can change the contents of an array, but cannot reassign the variable name to a new array or other data type. 


    let condiments = ['Ketchup', 'Mustard', 'Soy Sauce', 'Sriracha'];
    const utensils = ['Fork', 'Knife', 'Chopsticks', 'Spork'];
    condiments.push('Salt');
    console.log(condiments);
    utensils.pop();
    console.log(utensils) 
      
      [ 'Ketchup', 'Mustard', 'Soy Sauce', 'Sriracha', 'Salt' ]
      [ 'Fork', 'Knife', 'Chopsticks' ]

Using array methods we can mess with the data in the array, here the .push() method is used to add "Salt" to the condiments array and .pop() is used to remove the last element from the utensils array.

Let's use an array to handle Invoker's orbs and spells.

In my jsFunctions.html document I started writing a function that accepted a string input and determined which spell Invoker the DOTA 2 hero would invoke as well as another function that assigned the spell to the appropriate key based on what it was. The string that was used to interpret the correct spell was hard coded into the script, so let's use what we know about Arrays & Array.methods() to handle invoker's orbs. 


    //An array that holds the current orbs
    let orbs = ["quas", "wex", "exort"]
    /*Based on roll from 0-2 adds a new orb to the front of the array 
    and removes the last.*/
    let castOrb = () => {
      orbs.pop()
      let orbChoice = (Math.floor(Math.random() * 3))
      switch (orbChoice) {
        case 0:
          orbs.unshift("quas")
          break;
        case 1:
          orbs.unshift("wex")
          break;
        case 2:
          orbs.unshift("exort")
          break;
      }
    }
    console.log(orbs)
    castOrb()
    console.log(orbs)
    castOrb()
    console.log(orbs)
    castOrb()
    console.log(orbs)
      
    [ 'quas', 'wex', 'exort' ]
    [ 'quas', 'quas', 'wex' ]
    [ 'exort', 'quas', 'quas' ]
    [ 'exort', 'exort', 'quas' ]
