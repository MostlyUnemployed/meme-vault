Javascript Modules

Below is an example of how to define a module, and how to export it using the statement module.exports.

// ../1-airplane.js
    let Airplane = {};
Airplane.myAirplane = "G6 with a pepe sticker";
module.exports = Airplane;

Now that we've exported some code as a module to use elsewhere in our application we need can use the require() function, This allows us to access the exported module and the behavior we define within it. 

// Arrow Function Syntax
const Airplane = require ('./1-airplane.js')
let displayAirplane = () => {
console.log(Airplane.myAirplane)
};
displayAirplane()

//Other Syntax
const Airplane = require ('./1-airplane.js')
function displayAirplane() {
console.log(Airplane.myAirplane)
};
displayAirplane()
